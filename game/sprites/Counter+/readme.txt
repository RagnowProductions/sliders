# Counter+
Counter+ is a free counter software for video games.
## Accepted Digits
Counter+ supports integers (0-9), decimal symbols, and negative symbols
## Customization
In order to customize this tool, all you need to do is upload a file with a one character name (ex. j.svg, l.png, z.jpeg), then you can just type in the file name into the "Write digit (int) at (x) (y)" block (ex: write h at 144 -73), and it will display your number.
